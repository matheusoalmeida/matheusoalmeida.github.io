Eu sou Matheus Almeida, entusiasta e iniciante no desenvolvimento WEB com HTML, CSS e JavaScript.


Obrigado por acessar o meu portifolio, aqui estão alguns dos projeto que desenvolvi para demonstrar o meu conhecimento, para esses desenvolvimentos busquei utilizar temas com que estou familiarizado.

Meus projetos estão divididos em três categorias:


Landing Pages:

Páginas web focadas em converter visitantes em leads ou clientes, otimizadas para uma ação específica, como preencher um formulário ou fazer uma compra.


Páginas com uso de API's:

Páginas que utilizam APIs para obter ou enviar dados de fontes externas, como serviços de terceiros, em tempo real ou para funcionalidades como login ou pagamentos.


Páginas de tratamento de Dados:

Páginas dedicadas à recolha, processamento e visualização de dados, usadas para criar relatórios e análises através de gráficos e tabelas.
